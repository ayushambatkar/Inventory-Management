# Inventory Items Manager

Minimal full-stack app for managing inventory items using Express (backend) and React (frontend).

## What It Does

- View all inventory items
- Add a new item with name, sku, and quantity
- Show computed stock status: Out of Stock, Low Stock, or In Stock

## Project Structure

- backend
- frontend

## Run Locally

Prerequisite: Node.js installed.

1. Start backend

```bash
cd backend
npm install
npm start
```

2. Start frontend (in a second terminal)

```bash
cd frontend
npm install
npm run dev
```

3. Open the frontend URL shown in terminal (usually http://localhost:5173)

Backend runs on http://localhost:4000.
